## Status meanings

- **Proposed** — under discussion
- **Accepted** — adopted and reflected in the workspace
- **Deprecated** — no longer recommended
- **Superseded** — replaced by a newer ADR
- **Rejected** — considered but not adopted

## Conventions

- Keep `README.md` and `AGENTS.md` lean.
- Record significant workspace-level decisions here.
- Put detailed workflow documentation under `pipeline/docs/`.
